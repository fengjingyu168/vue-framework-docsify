# 项目样式

根据样式具体需要及历史原因，框架样式会出现在三个位置，具体为全局引入公用样式、样式配置文件引入、页面独立样式。

## 全局引入公用样式

第三方组件样式，框架抽离出的全局样式，均在main.js中引入。该部分样式将会应用到框架全局，而无需页面单独引人。

```js
import 'iview/dist/styles/iview.css'
```


## 样式配置文件引入

为规范使用样式，框架将样式指标独立配置，页面直接应用对应样式关键字实现样式添加。这个的好处是样式方便管理，可控。

```css
@color-blue: #0080FF;
@color-red: red;
@color-gray: #cccccc;
```
